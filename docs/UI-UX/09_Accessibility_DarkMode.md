# Accesibilidad + Dark Mode Checklist – v1

## Accesibilidad básica
- Touch targets ≥ 44px
- Labels en inputs (no solo placeholder)
- Focus visible en teclado
- Contraste AA mínimo
- Texto escalable (no romper layout)

## Dark mode
- Verificar contraste en:
  - texto secundario
  - chips
  - placeholders
  - estados disabled/locked

## Estados UI obligatorios por pantalla
- Loading (skeleton)
- Empty (guía + CTA)
- Error (reintentar)
- Success (toast)
