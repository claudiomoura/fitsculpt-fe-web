# FitSculpt — Investor Snapshot

FitSculpt integra entrenamiento, nutrición y motor adaptativo en un sistema único.

Diferenciación:
- Ajuste automático semanal
- Logging por voz e imagen
- Integración wearables
- Modelo explicable

Roadmap 12 meses:
- Adaptive Engine
- Voice + Vision Logging
- Health integrations
- Evaluación científica
