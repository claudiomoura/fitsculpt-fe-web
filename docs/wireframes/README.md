1. `/app/seguimiento` (Hub Diario) - See pencil-new.pen for visual
2. `/app/seguimiento/body-scan-report` (Body Scan Report) - See pencil-new.pen for visual

## Design Decisions

### Common Principles
- **Above the fold**: maximum 3 major blocks
- **One primary CTA** visible at a time
- **Maximum 1 chart** above the fold
- **Maximum 4 quick metrics** (chips)
- **Progressive disclosure** for details
- **Decision-first** in weekly review

### Color Scheme
- Primary CTA: `#3B82F6` (blue)
- Success/Accept: `#16A34A` (green)
- Warning/Action: `#FEF3C7` (yellow)
- Secondary/Info: `#F1F5F9` (light gray)
- Text Primary: `#1E293B` (dark)
- Text Secondary: `#64748B` (muted)

### Mobile Specifications
- Width: 375px (iPhone SE reference)
- Height: 812px
- Padding: 16px standard
- Corner radius: 8-12px for cards
